WhatsApp

Envoyer un WhatsApp sans ajouter le contact

3 min de lecture

Quand vous faites de la prospection multicanal, vous ne voulez pas polluer vos contacts avec des centaines de numéros de prospects. WhatsApp a une solution native que peu de gens connaissent : l'API wa.me.

Le principe

WhatsApp permet d'ouvrir une conversation directement via un lien, sans avoir besoin d'enregistrer le numéro dans vos contacts. C'est parfait pour :

  • Envoyer des messages de prospection
  • Faire des relances après un email ou un DM LinkedIn
  • Automatiser des séquences multicanal

Le format du lien

https://wa.me/{numéro}?text={message encodé} // Exemple concret : https://wa.me/33612345678?text=Bonjour%20Marc%2C%20suite%20%C3%A0%20mon%20message%20LinkedIn...

Les règles importantes :

  1. Format du numéro : international sans le + (33 pour France, pas 0033 ni +33)
  2. Encodage du message : les espaces deviennent %20, les accents sont encodés
  3. Pas de préfixe 0 : 612345678, pas 0612345678

Cas d'usage : la relance multicanal

Imaginons que vous avez envoyé un DM LinkedIn il y a 3 jours sans réponse. Vous avez le numéro du prospect (via Clay ou autre enrichissement). Voici comment structurer votre relance WhatsApp :

// Template de relance https://wa.me/33612345678?text=Salut%20{{prénom}}%20%F0%9F%91%8B%0A%0AJe%20me%20permets%20de%20te%20relancer%20ici%20suite%20%C3%A0%20mon%20message%20LinkedIn.%0A%0AOn%20peut%20en%20discuter%205%20min%20cette%20semaine%20%3F

Pro tip : Utilisez un outil comme urlencoder.org pour encoder vos messages avec accents et emojis.

Automatiser avec des variables

Si vous utilisez Clay, Make, ou n8n, vous pouvez générer ces liens dynamiquement avec les données de vos prospects :

// Formule Clay / Excel ="https://wa.me/"&A2&"?text=Salut%20"&B2&"%20..." // Où A2 = numéro, B2 = prénom

Attention aux limites

  • WhatsApp peut bloquer votre compte si vous envoyez trop de messages à des inconnus
  • Respectez un délai entre chaque message (minimum 30 secondes)
  • Variez vos messages pour éviter la détection de spam
  • Utilisez WhatsApp Business pour plus de crédibilité
